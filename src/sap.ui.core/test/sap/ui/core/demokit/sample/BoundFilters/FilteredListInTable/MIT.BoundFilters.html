Manual Integration Test: Bound Filters in a Table

DESCRIPTION

Test that the Bound Filters feature works correctly in an aggregation binding. The application shows a table of customers. Each row contains a dropdown (Select) whose list of account managers is filtered to only show managers responsible for the same region as the customer in that row.

Automated Tests: All manual tests described here are automated in Opa.qunit.js.

PREPARATION

To check console errors open the developer tools before starting the application and verify that there are no errors or warnings during any of the steps below.

Start the Bound Filters application. URL (adjust host and port if necessary):

  • as sample preview in the SDK: http://<host>:<port>/sapui5-sdk-dist/#/sample/sap.ui.core.sample.BoundFilters.FilteredListInTable/preview
  • local: http://<host>:<port>/test-resources/sap/ui/core/demokit/sample/common/index.html?component=BoundFilters.FilteredListInTable

REALIZE and CHECK

Initial State

Step Action Expected Result
1 Open the application. A table titled "Customers" is displayed with 10 visible rows. Each row shows a customer name, a region, and a Select control.

Bound Filters: Americas region (first row)

The first row shows customer TechCorp Solutions with region Americas. There are 5 account managers responsible for the Americas region: John Smith, Sarah Johnson, Mike Williams, Jennifer Brown, and David Jones.

Step Action Expected Result
1 Click on the Select dropdown in the first row (TechCorp Solutions, Americas). The dropdown opens and shows exactly 5 entries: John Smith, Sarah Johnson, Mike Williams, Jennifer Brown, David Jones. No account managers from other regions are shown.
2 Close the dropdown (press Escape or click elsewhere). The dropdown closes.

Bound Filters: EMEA region

Find a row with region EMEA (e.g. Global Industries Ltd, row 3). There are 4 account managers for EMEA: Emma Anderson, Lucas Mueller, Sophie Dubois, and Marco Rossi.

Step Action Expected Result
1 Click on the Select dropdown in a row with region EMEA. The dropdown shows exactly 4 entries: Emma Anderson, Lucas Mueller, Sophie Dubois, Marco Rossi. No Americas or APJ managers are shown.
2 Close the dropdown. The dropdown closes.

Bound Filters: APJ region

Find a row with region APJ (e.g. Asia Pacific Ventures, row 4). There are 4 account managers for APJ: Yuki Tanaka, Raj Patel, Li Chen, and Priya Sharma.

Step Action Expected Result
1 Click on the Select dropdown in a row with region APJ. The dropdown shows exactly 4 entries: Yuki Tanaka, Raj Patel, Li Chen, Priya Sharma. No Americas or EMEA managers are shown.
2 Close the dropdown. The dropdown closes.
