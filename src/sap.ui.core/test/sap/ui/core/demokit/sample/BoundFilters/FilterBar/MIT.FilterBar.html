Manual Integration Test: Bound Filters in a Filter Bar

DESCRIPTION

Test that the Bound Filters feature works correctly in a filter bar scenario. The application shows a table of 15 team members with a filter bar toolbar. Filters are split into two groups: Organizational (department prefix, location prefix) and Personal (first name prefix, last name prefix). A "Show Personal Filters" / "Show Organizational Filters" button toggles between the two groups. Switching groups clears the now-hidden group's filter values so they become inactive. All visible filters are applied instantly when a value is entered.

Automated Tests: All manual tests described here are automated in Opa.qunit.js.

PREPARATION

To check console errors open the developer tools before starting the application and verify that there are no errors or warnings during any of the steps below.

Start the Bound Filters application. URL (adjust host and port if necessary):

  • as sample preview in the SDK: http://<host>:<port>/sapui5-sdk-dist/#/sample/sap.ui.core.sample.BoundFilters.FilterBar/preview
  • local: http://<host>:<port>/test-resources/sap/ui/core/demokit/sample/common/index.html?component=BoundFilters.FilterBar

REALIZE and CHECK

Initial State

Step Action Expected Result
1 Open the application. A table titled "Employees" is displayed. The filter bar shows the organizational filters: Department prefix and Location prefix inputs are empty. The "Show Personal Filters" button is visible on the right. All 15 team members are shown.

Instant Filter: Department prefix & Location prefix

Step Action Expected Result
1 Type "Dev" in the Department prefix input. Press Enter. The table instantly updates and shows 7 rows: all Development employees.
2 Type "Wal" in the Location prefix input. Press Enter. The table instantly updates and shows 2 rows: all Development employees located in Walldorf.

Instant Filter: First name prefix & Last name prefix

Step Action Expected Result
1 Click "Show Personal Filters". The organizational filter inputs (Department prefix, Location prefix) are hidden and the personal filter inputs (First name prefix, Last name prefix) appear. All 15 rows are shown. The button now reads "Show Organizational Filters".
2 Type "J" in the First name prefix input. Press Enter. The table shows 4 rows with first names starting with "J".
3 Type "S" in the Last name prefix input. Press Enter. The table shows 1 rows with first names starting with "J" and last names starting with "S".
